## ⚠️ 常见问题排查

### Q: npm install 失败
```bash
# 手动使用 sudo 权限安装
sudo npm install
sudo chown -R $(whoami):staff node_modules
```

### Q: 端口被占用
```bash
# 查找并杀死占用进程
lsof -i :9000
kill -9 <PID>
```

### Q: 仍然出现 vue-router 错误
```bash
# 确保 package.json 有 vue-router
grep "vue-router" package.json

# 如果没有，手动添加
npm install vue-router@4.3.0 --save
```
