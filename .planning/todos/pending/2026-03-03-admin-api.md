## Problem

## Solution

按功能模块拆分到 modules/ 目录：
- modules/user.js - 用户管理
- modules/coupon.js - 优惠券管理
- modules/banner.js - Banner管理
- modules/inventory.js - 库存管理
- modules/supplier.js - 供应商管理
- modules/purchase.js - 采购管理
