## 发现的潜在问题

### 1. 旧测试文件格式问题
- `test-v3.test.js` 和 `test.test.js` 使用 describe 语法
- 与当前测试运行器不兼容

### 2. 常量值不一致
- `test-v3.test.js` 中的晋升门槛值与 `common/constants.js` 中的定义不一致
