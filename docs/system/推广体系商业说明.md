# 大友元气推广体系 - 商业说明文档

> **面向对象**: 公司管理层、运营团队
> **更新时间**: 2026年2月

---

## 一、这个系统是什么？

简单来说，这是一个**"全民推广，多劳多得"**的会员分销系统。

任何人都可以成为我们的推广员，通过分享商品链接赚取佣金。推广得越多，级别越高，赚的也越多。

---

## 二、两种身份，两套晋升通道

我们的推广员有**两套身份**，互不影响，都能升级：

### 身份一：代理等级（看"关系"）

根据你是谁推荐进来的，决定你的代理等级：

```
总公司
   ↓
一级代理（直接和总公司合作的大代理）
   ↓
二级代理（一级代理推荐的人）
   ↓
三级代理（二级代理推荐的人）
   ↓
四级代理（普通用户，刚加入的新人）
```

**简单理解**：越早加入、离总公司越近，基础佣金越高。

| 代理等级 | 基础佣金 | 举例（卖出100元） |
| -------- | -------- | ----------------- |
| 一级代理 | 10%      | 赚 10 元          |
| 二级代理 | 6%       | 赚 6 元           |
| 三级代理 | 3%       | 赚 3 元           |
| 四级代理 | 1%       | 赚 1 元           |

---

### 身份二：星级身份（看"业绩"）

不管你的代理等级是什么，只要业绩达标，就能升级星级：

```
普通会员（刚加入）
    ↓
铜牌推广员（有业绩了）
    ↓
银牌推广员（业绩更好了）
    ↓
金牌推广员（销售冠军）
```

**简单理解**：卖得越多、团队越大，星级越高，能拿的奖励种类越多。

| 星级 | 需要达到的条件              | 解锁的额外奖励      |
| ---- | --------------------------- | ------------------- |
| 铜牌 | 累计卖货2万元 或 直推30人   | **复购奖 3%** |
| 银牌 | 本月卖货5万元 或 团队50人   | **管理奖 2%** |
| 金牌 | 本月卖货10万元 或 团队200人 | 更多权益（待开发）  |

### 佣金池怎么分？（举例说明）

假设推广链是这样的：

```
一级代理（张三）
    ↓ 推荐了
二级代理（李四）
    ↓ 推荐了
三级代理（王五）
    ↓ 推荐了

    ↓ 推荐了
```

**20元佣金池分配**：

| ------ | -------- | -------- | ------------------------- |

| 张三   | 一级     | 10元     | -                         |

**注意**：不是每个人都能拿满所有奖励，要看星级身份！

## 四、四种奖励类型详解

### 1️⃣ 基础佣金 - 人人都有

只要你在这个推广链上，就能拿基础佣金。

- 一级代理拿 10%
- 二级代理拿 6%
- 三级代理拿 3%
- 四级代理拿 1%

**条件**：无（只要被系统识别为推广链上的一员）

---

### 2️⃣ 复购奖励 - 留住老客户

当你的下级客户**第二次、第三次购买**时，你额外拿 3%。

**目的**：鼓励推广员维护好客户关系，不是卖一次就不管了。

**条件**：

- 你的星级 ≥ 铜牌
- 这单是老客户的复购

**例子**：小明去年在你这儿买过啤酒，今年又来买，这单就算复购单。

---

### 3️⃣ 团队管理奖 - 带团队有奖励

你带的人越多、团队业绩越好，你能拿管理奖。

**条件**：

- 你的星级 ≥ 银牌
- 你下面有团队在帮你卖货

**怎么算（级差制）**：

假设你有3个下级代理：A、B、C

```
你（银牌）
 ├── A（银牌）也能拿管理奖
 │    └── 客户下单
 ├── B（铜牌）不能拿管理奖
 │    └── 客户下单
 └── C（银牌）也能拿管理奖
      └── 客户下单
```

**分配规则**：管理奖总共2%，从最下面往上分：

- 如果下级A已经拿了2%管理奖 → 你就拿0%
- 如果下级B拿不了管理奖 → 你可以拿2%

**目的**：防止重复发奖，同时鼓励你培养下级成为银牌（因为他们升级后，你能拿育成津贴）。

---

### 4️⃣ 育成津贴 - 教会徒弟，师傅有钱

如果你培养的下级升级成为银牌推广员，你成为他的"导师"，以后他每赚一笔钱，你都能额外拿 2%。

**条件**：

- 你的下级升级到银牌
- 系统自动记录你为他的导师

**例子**：

```
你
 └── 小王（你培养的，现在是银牌）
       └── 小王卖出商品，赚了100元佣金
                 ↓
           你额外拿 2元 育成津贴
```

**目的**：打破"下级升级后上级没钱拿"的尴尬，鼓励上级真心帮助下级成长。

## 五、怎么升级星级？

### 升铜牌推广员（任选一条）

| 条件                | 达成方式                 |
| ------------------- | ------------------------ |
| 累计销售额 ≥ 2万元 | 卖出200单100元的啤酒     |
| 或 直推人数 ≥ 30人 | 直接推荐30个人注册小程序 |

### 升银牌推广员（任选一条）

| 条件                | 达成方式                     |
| ------------------- | ---------------------------- |
| 本月销售额 ≥ 5万元 | 这个月卖出500单100元的啤酒   |
| 或 团队人数 ≥ 50人 | 你和你的下级们总共发展了50人 |

### 升金牌推广员（任选一条）

| 条件                 | 达成方式                      |
| -------------------- | ----------------------------- |
| 本月销售额 ≥ 10万元 | 这个月卖出1000单100元的啤酒   |
| 或 团队人数 ≥ 200人 | 你和你的下级们总共发展了200人 |

**重要**：销售额按月重算，每个月1号从头开始统计！

## 六、公司能赚多少钱？

### 每笔订单的固定分配

```
订单金额 100%
    │
    ├── 公司固定拿 80%（用于成本、运营、利润）
    │
    └── 代理佣金池 20%（分给推广员）
```

**公司利润 = 80% - 商品成本 - 运营成本**

### 代理佣金池会超支吗？

**理论上可能**：当多个奖励同时触发时（复购+管理+育成），发放总额可能超过20%。

**实际控制手段**：

1. **级差制**：管理奖用级差算法，不会重复发
2. **星级门槛**：不是每个人都能拿复购奖和管理奖
3. **系统监控**：后台实时监控拨比，异常会报警

**运营建议**：定期查看 `reward_records` 表，分析实际拨比情况。

## 七、这个系统的好处

### 对公司的好处

1. **低成本获客**：只有成交才付佣金，不像广告要提前花钱
2. **裂变快**：推广员会主动拉更多推广员
3. **可控拨比**：80%固定归公司，佣金池封顶20%
4. **防作弊**：系统有多层反欺诈机制

### 对推广员的好处

1. **零门槛加入**：任何人都能成为推广员
2. **多劳多得**：卖得越多、团队越大，收入越高
3. **被动收入**：培养好下级后，躺着也能赚钱
4. **公平晋升**：星级只看业绩，不看关系

### 对消费者的好处

1. **正品保证**：通过推广员买的也是官方正品
2. **可能优惠**：推广员为了冲业绩可能会给返利
3. **也能赚钱**：买完觉得好，自己也能当推广员

## 八、常见问题

### Q1：我可以同时拿四种奖励吗？

**可以**，但前提是：

- 基础佣金：一定有
- 复购奖：要是铜牌，且这单是复购
- 管理奖：要是银牌，且用级差制计算
- 育成津贴：要是某人的导师

### Q2：我的代理等级能升吗？

**不能**。代理等级是"命中注定"的，由推荐你的人的等级决定。

- 一级代理推荐的人 → 二级代理
- 二级代理推荐的人 → 三级代理
- 以此类推

**但不用担心**：星级身份是靠业绩升级的，四级代理也能靠业绩升到金牌，拿更多奖励！

### Q3：下级升级后我就赚不到他的钱了吗？

**不会！** 这正是"育成津贴"存在的意义。

- 你的下级升级到银牌后，系统自动记录你是他的导师
- 以后他每赚一笔佣金，你都额外拿2%育成津贴
- **教会徒弟，师傅有钱拿**

### Q4：客户退单了，佣金怎么办？

系统会自动处理：

- 待结算状态的奖励会被取消
- 已结算的奖励会从账户扣除

### Q5：怎么提现？

推广员的钱有 **7天冻结期**，7天后自动转入可提现余额，随时可以申请提现。

## 十、总结

### 一句话概括

> **公司拿80%，代理分20%。推广越多，级别越高，奖励越多！**

### 核心逻辑

```
客户下单 → 公司拿80% → 代理佣金池20% → 按规则分给推广链
                                              ↓
                               基础佣金 + 复购奖 + 管理奖 + 育成津贴
```

### 升级路径

```
新用户注册
    ↓
开始推广，拿基础佣金
    ↓
业绩达标 → 升铜牌 → 解锁复购奖3%
    ↓
继续努力 → 升银牌 → 解锁管理奖2% + 培养下级拿育成津贴2%
    ↓
团队做大 → 升金牌 → 享受最高权益
```
