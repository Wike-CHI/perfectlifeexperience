# 大友元气推广体系 - 商业说明

**面向对象**: 公司管理层、运营团队、推广员
**更新时间**: 2026年2月

---

## 一、系统是什么？

全民推广，多劳多得。任何人分享商品链接都能赚佣金，推荐越多赚得越多。

**核心规则**：

- 每笔订单的20%作为佣金，分给推广员和上级
- 公司拿80%，代理分20%

---

## 二、等级体系

只有一套身份，两个名字：


| 内部等级 | 对外名称   | 被谁推荐       |
| -------- | ---------- | -------------- |
| 一级代理 | 金牌推广员 | 被总公司推荐   |
| 二级代理 | 银牌推广员 | 被一级代理推荐 |
| 三级代理 | 铜牌推广员 | 被二级代理推荐 |
| 四级代理 | 普通会员   | 被三级代理推荐 |

**推荐链示例**：

```
总公司
  ↓ 推荐
金牌（一级/张三）
  ↓ 推荐
银牌（二级/李四）
  ↓ 推荐
铜牌（三级/王五）
  ↓ 推荐
普通（四级/赵六）
```

**关键**：

- 等级由推荐人决定
- 等级越高，推广时拿得越多

---

## 三、佣金怎么分？

### 核心规则

**自己推广 = 和上级按固定金额分佣金**

### 示例1：一级推广（独拿）

**A（金牌）推荐客户**：

```
A推荐客户 → A独拿20元佣金
```

**分配**：

- A拿：20元

---

### 示例2：二级推广（2人分）

**B（银牌）推荐客户**：

```
B推荐客户 → B和A分20元
```

**分配**：

- B拿：**12元**
- A拿：**8元**
- 总计：20元 ✅

---

### 示例3：三级推广（3人分）

**C（铜牌）推荐客户**：

```
C推荐客户 → C、B、A三人分20元
```

**分配**：

- C拿：**12元**
- B拿：**4元**
- A拿：**4元**
- 总计：20元 ✅

---

### 示例4：四级推广（4人分）

**D（普通）推荐客户**：

```
D推荐客户 → D、C、B、A四人分20元
```

**分配**：

- D拿：**8元**
- C拿：**4元**
- B拿：**4元**
- A拿：**4元**
- 总计：20元 ✅

---

## 四、佣金分配规则总结


| 推广人等级 | 自己拿 | 一级拿 | 二级拿 | 三级拿 | 总计 |
| ---------- | ------ | ------ | ------ | ------ | ---- |
| 一级推广   | 20元   | -      | -      | -      | 20元 |
| 二级推广   | 12元   | 8元    | -      | -      | 20元 |
| 三级推广   | 12元   | 4元    | 4元    | -      | 20元 |
| 四级推广   | 8元    | 4元    | 4元    | 4元    | 20元 |

**规律**：

- 一级推广：独拿20元
- 二级推广：自己拿12元，上级拿8元
- 三级推广：自己拿12元，上级各拿4元
- 四级推广：自己拿8元，上级各拿4元

## 五、可以升级！

### 升级条件


| 从   | 升到 | 条件（满足其一）            |
| ---- | ---- | --------------------------- |
| 普通 | 铜牌 | 累计卖货2万元               |
| 铜牌 | 银牌 | 本月卖货5万元 或 团队50人   |
| 银牌 | 金牌 | 本月卖货10万元 或 团队200人 |

**注意**：

- 销售额每月1号重算
- 团队人数 = 你推荐的所有人（包括间接推荐）

---

### 升级后发生什么？

**下级升级 = 脱离原推荐链，可以发展下线**

#### 示例1：D（四级）升级到三级

**升级前**：

```
A（金牌） → B（银牌） → C（铜牌） → D（普通） → 客户
D推荐客户，佣金分配：
  - D：8元
  - C：4元
  - B：4元
  - A：4元
```

**升级后**：

```
A（金牌） → B（银牌） → D（铜牌） → 客户
                      ↓
                   C（铜牌） → 新下级E（四级） → 其他客户
```

**D推荐客户，佣金分配**：

- D拿：12元（升级后自己拿更多！）✅
- B拿：4元
- A拿：4元
- C拿：0元（C失去了D）

**C怎么办？**

- C发展新的下级E（四级）
- E推荐客户时：
  - E拿：8元
  - C拿：4元
  - B拿：4元
  - A拿：4元

---

#### 示例2：D（三级）继续升级到二级

**升级后**：

```
A（金牌） → D（银牌） → 客户
         ↓
      B（银牌） → C（铜牌） → 其他客户
```

**D推荐客户，佣金分配**：

- D拿：12元
- A拿：8元
- 总计：20元

**D的好处**：

- 从三级升级到二级
- 自己推广时从拿12元（和上级分）变成拿12元（但只有一级拿8元）
- 可以发展下线三级、四级

---

#### 示例3：C（三级）升级到二级

**升级前**：

```
A（金牌） → B（银牌） → C（铜牌） → D/E/F（四级）
```

**升级后**：

```
A（金牌） → B（银牌） → C（银牌） → 新下级（三级） → 新下级（四级）
```

**C的变化**：

- 脱离B的推荐链
- 自己推广时拿12元，A拿8元
- 可以发展自己的下线

---

## 六、为什么要培养下级升级？

### 短期看：上级吃亏

- 下级升级前：上级拿下级的佣金
  - 四级推广时，C拿4元
- 下级升级后：上级失去这个下级
  - 四级升级后，C拿0元

### 长期看：这是自然淘汰机制

1. **公司要求**：上级要升级，必须培养X个下级升级
2. **新陈代谢**：下级成长是必然的，上级需要不断发展新下级
3. **良性循环**：能力强的人升级，能力弱的人被淘汰

### 培养下级的好处

虽然失去一个下级，但：

1. **完成自己的升级条件**
2. **腾出位置发展新下级**
3. **整体团队能力提升**

## 八、为什么这样设计？

### 对公司

✅ 成交才付佣金，无风险
✅ 裂变式获客
✅ 80%固定收益
✅ 自动淘汰弱者，保留强者

### 对代理

✅ 零门槛加入（任何人都可以从四级开始）
✅ 多劳多得
✅ 升级后佣金更高（从8元到12元）
✅ 四级就拿8元，有推广动力

### 对消费者

✅ 正品保证
✅ 买完觉得好，自己也能赚钱

## 九、常见问题

**Q：我升级后，我的推荐人还能赚我的钱吗？**
不能。你升级后脱离原推荐链，直接和上上级对接。

**Q：下级升级后，我就赚不到他的钱了吗？**
对。下级升级后脱离你的推荐链，你失去这个下级的佣金。

**Q：那我为什么要培养下级升级？**
因为：

1. 公司要求你培养X个下级升级，你才能升级
2. 下级升级是必然的，你接受这个机制
3. 你可以不断发展新下级来补充

**Q：退单了怎么办？**
待结算佣金取消，已结算佣金扣除。

**Q：怎么提现？**
7天冻结期后可随时提现。

**Q：我最多能拿多少佣金？**
单笔订单最多拿20元（20%），自己推荐。

**Q：四级代理拿8元，有动力吗？**
有！8元已经是订单金额的8%，而且升级后可以拿12元。

## 十、总结

**一句话**：

> 公司拿80%，代理分20%。等级越高，自己推广时拿得越多！

**核心逻辑**：

```
客户下单 → 佣金池20元 → 按等级分配
                            ↓
         一级独拿20元 或 多人按固定金额分
```

**分配规则**：

```
一级推广：自己拿20元
二级推广：自己12元，一级8元
三级推广：自己12元，二级4元，一级4元
四级推广：自己8元，三级4元，二级4元，一级4元
```

**升级路径**：

```
普通（四级，8元） → 铜牌（三级，12元） → 银牌（二级，12元） → 金牌（一级，20元）
```

**记住**：

- 等级越高，自己推广时拿得越多
- 下级升级 = 失去下级
- 不断发展新下级 = 收入持续增长
