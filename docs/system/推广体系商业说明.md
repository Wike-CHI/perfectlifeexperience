# 大友元气推广体系 - 商业说明

**面向对象**: 公司管理层、运营团队、推广员
**更新时间**: 2026年2月

---

## 一、系统是什么？

全民推广，多劳多得。任何人分享商品链接都能赚佣金，推荐越多赚得越多。

**核心规则**：

- 每笔订单的20%作为佣金，分给推广员和上级
- 公司拿80%，代理分20%

---

## 二、等级体系

只有一套身份，两个名字：


| 内部等级 | 对外名称   | 被谁推荐       |
| -------- | ---------- | -------------- |
| 一级代理 | 金牌推广员 | 被总公司推荐   |
| 二级代理 | 银牌推广员 | 被一级代理推荐 |
| 三级代理 | 铜牌推广员 | 被二级代理推荐 |
| 四级代理 | 普通会员   | 被三级代理推荐 |

**推荐链示例**：

```
总公司
  ↓ 推荐
金牌（一级/张三）
  ↓ 推荐
银牌（二级/李四）
  ↓ 推荐
铜牌（三级/王五）
  ↓ 推荐
普通（四级/赵六）
```

**关键**：

- 等级由推荐人决定
- 等级越高，推广时拿得越多

---

## 三、佣金怎么分？

### 核心规则

**自己推广 = 和上级按固定金额分佣金**

### 示例1：一级推广（独拿）

**A（金牌）推荐客户**：

```
A推荐客户 → A独拿20元佣金
```

**分配**：

- A拿：20元

---

### 示例2：二级推广（2人分）

**B（银牌）推荐客户**：

```
B推荐客户 → B和A分20元
```

**分配**：

- B拿：**12元**
- A拿：**8元**
- 总计：20元 ✅

---

### 示例3：三级推广（3人分）

**C（铜牌）推荐客户**：

```
C推荐客户 → C、B、A三人分20元
```

**分配**：

- C拿：**12元**
- B拿：**4元**
- A拿：**4元**
- 总计：20元 ✅

---

### 示例4：四级推广（4人分）

**D（普通）推荐客户**：

```
D推荐客户 → D、C、B、A四人分20元
```

**分配**：

- D拿：**8元**
- C拿：**4元**
- B拿：**4元**
- A拿：**4元**
- 总计：20元 ✅

## 四、佣金分配规则总结


| 推广人等级 | 自己拿 | 一级拿 | 二级拿 | 三级拿 | 总计 |
| ---------- | ------ | ------ | ------ | ------ | ---- |
| 一级推广   | 20元   | -      | -      | -      | 20元 |
| 二级推广   | 12元   | 8元    | -      | -      | 20元 |
| 三级推广   | 12元   | 4元    | 4元    | -      | 20元 |
| 四级推广   | 8元    | 4元    | 4元    | 4元    | 20元 |

**规律**：

- 一级推广：独拿20元
- 二级推广：自己拿12元，上级拿8元
- 三级推广：自己拿12元，上级各拿4元
- 四级推广：自己拿8元，上级各拿4元

## 五、可以升级！

### 升级条件


| 从   | 升到 | 条件（满足其一）            |
| ---- | ---- | --------------------------- |
| 普通 | 铜牌 | 累计卖货2万元               |
| 铜牌 | 银牌 | 本月卖货5万元 或 团队50人   |
| 银牌 | 金牌 | 本月卖货10万元 或 团队200人 |

**注意**：

- 销售额每月1号重算
- 团队人数 = 你推荐的所有人（包括间接推荐）

---

### 升级后发生什么？

**核心规则：下级升级 = 脱离原推荐链，但有例外！**

有两种升级情况：
1. **自己升级脱离**：下级自己满足条件升级，脱离原推荐链
2. **跟随上级升级**：上级升级后，原直接下级可以跟随升级

---

### 跟随升级机制（重要！）

当上级升级时，他原来的直接下级可以**跟随升级**，这是"下级升级脱离"的例外情况。

| 上级升级 | 下级跟随升级规则 |
|---------|----------------|
| **4→3** | 无下级跟随（但可发展新的四级） |
| **3→2** | 之前带的4级 → 3级 |
| **2→1** | 之前带的3级→2级，4级→3级 |

**关键点**：跟随升级后，下级会脱离原推荐链，成为上级的上级或平级。

---

#### 示例1：D（四级）升级到三级（自己升级）

**升级前**：

```
A（金牌） → B（银牌） → C（铜牌） → D（普通） → 客户
D推荐客户，佣金分配：
  - D：8元
  - C：4元
  - B：4元
  - A：4元
```

**升级后**：

```
A（金牌） → B（银牌） → D（铜牌） → 客户
                      ↓
                   C（铜牌） → 新下级E（四级） → 其他客户
```

**D推荐客户，佣金分配**：

- D拿：12元（升级后自己拿更多！）✅
- B拿：4元
- A拿：4元
- C拿：0元（C失去了D）

**C怎么办？**

- C发展新的下级E（四级）
- E推荐客户时：
  - E拿：8元
  - C拿：4元
  - B拿：4元
  - A拿：4元

---

#### 示例2：D（三级）升级到二级，带动C升级（跟随升级）

**升级前**：

```
A（金牌） → B（银牌） → C（铜牌） → D（铜牌）
                              ↓
                           E（四级）
```

**D升级到二级后**：

```
A（金牌） → B（银牌） → D（银牌） → 新下级（三级/四级）
         │         ↓
         │      C（银牌） → E（三级） → 其他客户
         │
         └─其他三级下级
```

**重要变化**：

1. **D脱离C**，直接成为B的下级
2. **C跟随D升级到二级**（因为是D原来的直接下级）
3. **C脱离B**，直接成为A的下级
4. **E跟随C升级到三级**（因为是C原来的直接下级）

**佣金变化**（D推荐100元订单）：

| 人员 | 升级前 | 升级后 | 变化 |
|------|--------|--------|------|
| D | 12元 | 12元 | 金额不变，但上级从多人变成一人（只A拿8元） |
| C | 4元 | 0元 | 失去D，但自己升级后推广拿12元 |
| B | 4元 | 0元 | 失去C |
| A | 4元 | 8元 | 直接上级D |

**C的新权益**：

- 自己推广时从"12元+B拿4元"变成"12元+A拿8元"
- 可以发展自己的下线（E从四级升到三级）

---

#### 示例3：D（二级）升级到一级，带动C和E升级（跟随升级）

**升级前**：

```
A（金牌） → B（银牌） → C（银牌） → E（三级） → F（四级）
         │         ↓
         │      D（银牌） → 其他客户
         │
         └─其他下级
```

**D升级到一级后**：

```
D（金牌） → 新二级 → 新三级 → 新四级

A（金牌） → B（银牌） → C（金牌） → E（二级） → F（三级）
         │
         └─其他下级
```

**重要变化**：

1. **D脱离原推荐链**，成为独立的总公司
2. **C跟随D升级到一级**（因为是D原来的直接下级）
3. **E跟随C升级到二级**（因为是C原来的直接下级）
4. **F跟随E升级到三级**（因为是E原来的直接下级）

**佣金变化**（D推荐100元订单）：

| 人员 | 升级前 | 升级后 | 变化 |
|------|--------|--------|------|
| D | 12元 | 20元 | 独拿！提升67% |
| A | 8元 | 0元 | 失去D |
| B | 0元 | 0元 | 无变化 |
| C | 自己推广12元 | 自己推广20元 | 跟随升级到一级 |

**跟随升级的好处**：

- 下级可以快速升级，不需要重新满足升级条件
- 团队整体等级提升，推广动力更强
- 形成良性循环：上级升级 → 下级跟随 → 再发展新下级

---

### 自己升级 vs 跟随升级对比

| 升级类型 | 条件 | 推荐关系变化 | 下级是否跟随 |
|---------|------|-------------|------------|
| **自己升级** | 满足业绩/人数条件 | 脱离原推荐链，跳级对接上上级 | 否 |
| **跟随升级** | 上级升级触发 | 脱离原推荐链，成为上级的上级/平级 | 是（原直接下级） |

**示例对比**：

假设 C 是 D 的上级

- **D自己升级**：D升到二级后，C失去D，C的等级不变
- **D带动C升级**：D升到一级后，C跟随升到一级（因为C是D原来的直接下级）

## 六、为什么要培养下级升级？

### 短期看：上级吃亏

- 下级升级前：上级拿下级的佣金
  - 四级推广时，C拿4元
- 下级升级后：上级失去这个下级
  - 四级升级后，C拿0元

### 长期看：这是自然淘汰机制

1. **公司要求**：上级要升级，必须培养X个下级升级
2. **新陈代谢**：下级成长是必然的，上级需要不断发展新下级
3. **良性循环**：能力强的人升级，能力弱的人被淘汰

### 培养下级的好处

**下级自己升级：**
虽然失去一个下级，但：
1. **完成自己的升级条件**
2. **腾出位置发展新下级**
3. **整体团队能力提升**

**上级升级带动下级（跟随升级）：**
不仅不吃亏，反而双赢：
1. **上级升级**：自己推广时佣金更高
2. **下级跟随升级**：不需要重新满足条件，直接升级
3. **团队整体提升**：形成良性循环
4. **关系更紧密**：下级感恩上级的带动

**示例**：
- D从二级升到一级
- C（D的原下级）跟随升到一级
- E（C的原下级）跟随升到二级
- F（E的原下级）跟随升到三级
- **整个团队等级都提升了！**

## 九、常见问题

**Q：我升级后，我的推荐人还能赚我的钱吗？**
**A：** 分两种情况：
- **自己升级**：不能。脱离原推荐链，直接和上上级对接
- **跟随上级升级**：可以。上级升级带动你升级，你成为上级的上级或平级，关系更紧密

**Q：下级升级后，我就赚不到他的钱了吗？**
**A：** 分两种情况：
- **下级自己升级**：对。下级脱离你的推荐链，你失去这个下级的佣金
- **你升级带动下级**：对，但双赢！你升级后佣金更高，下级也跟着升级了

**Q：那我为什么要培养下级升级？**
**A：** 短期看确实会失去下级，但长期看：
1. 公司要求你培养X个下级升级，你才能升级
2. 下级升级是必然的，你接受这个机制
3. **你升级时可以带动下级**，形成团队共赢
4. 不断发展新下级 = 收入持续增长

**Q：退单了怎么办？**
待结算佣金取消，已结算佣金扣除。

**Q：怎么提现？**
7天冻结期后可随时提现。

**Q：我最多能拿多少佣金？**
单笔订单最多拿20元（20%），自己推荐。

**Q：四级代理拿8元，有动力吗？**
有！8元已经是订单金额的8%，而且升级后可以拿12元。

**Q：什么是跟随升级？**
**A：** 当上级升级时，他原来的直接下级可以跟随升级：
- 上级4→3：无下级跟随
- 上级3→2：之前带的4级可升到3级
- 上级2→1：之前带的3级→2级，4级→3级

**Q：跟随升级后，推荐关系怎么变？**
**A：** 跟随升级后，下级会脱离原推荐链：
- 原关系：A→B→C→D
- B升级带动C：C脱离B，可能成为A的下级或独立
- 形成新的、更高级的推荐关系

## 十、总结

**一句话**：

> 公司拿80%，代理分20%。等级越高，自己推广时拿得越多！

**核心逻辑**：

```
客户下单 → 佣金池20元 → 按等级分配
                            ↓
         一级独拿20元 或 多人按固定金额分
```

**分配规则**：

```
一级推广：自己拿20元
二级推广：自己12元，一级8元
三级推广：自己12元，二级4元，一级4元
四级推广：自己8元，三级4元，二级4元，一级4元
```

**升级路径**：

```
普通（四级，8元） → 铜牌（三级，12元） → 银牌（二级，12元） → 金牌（一级，20元）
```

**跟随升级机制**：

```
上级3→2：带4级升3级
上级2→1：带3级升2级，4级升3级
形成团队共赢！
```
```

**记住**：

- 等级越高，自己推广时拿得越多
- 下级自己升级 = 失去下级（短期损失）
- **上级升级 = 带动下级跟随升级（团队共赢）**
- 不断发展新下级 = 收入持续增长
- 不断发展新下级 = 收入持续增长
