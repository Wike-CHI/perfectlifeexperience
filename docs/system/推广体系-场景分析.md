# 推广体系场景分析

## 基础假设

- 佣金池封顶：20%
- 代理池：100人
- 等级：金牌（10%）、银牌（6%）、铜牌（3%）、普通（1%）

---

## 场景1：简单推荐链，无升级

```
金牌A → 银牌B → 铜牌C → 普通D → 客户
```

**100元订单佣金分配**：
| 等级 | 代理人 | 佣金 |
|------|--------|------|
| 金牌 | A | 10元 |
| 银牌 | B | 6元 |
| 铜牌 | C | 3元 |
| 普通 | D | 1元 |
| **总计** | | **20元** ✅ |

---

## 场景2：普通D升级到铜牌

### 升级前（D卖货100元）

| 等级 | 代理人 | 佣金 |
|------|--------|------|
| 金牌 | A | 10元 |
| 银牌 | B | 6元 |
| 铜牌 | C | 3元 |
| 普通 | D | 1元 |
| **总计** | | **20元** |

### 升级后（D卖货100元）

**方案A：按推荐链分 + 育成津贴（额外给）**

| 等级 | 代理人 | 基础佣金 | 育成津贴 | 总计 |
|------|--------|----------|----------|------|
| 铜牌 | D | 3元 | - | 3元 |
| 铜牌 | C | 3元 | 0.06元（3×2%） | 3.06元 |
| 银牌 | B | 6元 | - | 6元 |
| 金牌 | A | 10元 | - | 10元 |
| **总计** | | **22元** | **0.06元** | **22.06元** ❌ |

**问题**：超支2.06元，公司只拿77.94元！

---

**方案B：育成津贴从D的佣金扣**

| 等级 | 代理人 | 实际佣金 |
|------|--------|----------|
| 铜牌 | D | 3 - 0.06 = 2.94元 |
| 铜牌 | C | 3元 |
| 银牌 | B | 6元 |
| 金牌 | A | 10元 |
| **总计** | | **20元** ✅ |

**问题**：D升级后反而拿得更少（从1元变成2.94元？不对，是从1元变成3元，但要被导师抽0.06元）

---

**方案C：推荐链不变，育成津贴从20%池里扣**

| 等级 | 代理人 | 计算方式 | 佣金 |
|------|--------|----------|------|
| 金牌 | A | 20 × (10/20) = 10元 | 10元 |
| 银牌 | B | 20 × (6/20) = 6元 | 6元 |
| 铜牌 | C | 20 × (3/20) = 3元 | 3元 |
| 铜牌 | D | 20 × (1/20) = 1元 | 1元 |
| 育成津贴 | C | D佣金 × 2% = 1 × 0.02 = 0.02元 | 0.02元 |
| **总计** | | | **20.02元** ❌ |

**问题**：还是超支0.02元

---

**方案D：重新定义推荐链佣金**

下级升级后，推荐链的佣金按当前等级计算：

| 等级 | 代理人 | 逻辑 | 佣金 |
|------|--------|------|------|
| 金牌 | A | 最高级 | 10元 |
| 银牌 | B | 第二级 | 6元 |
| 铜牌 | C | D的导师 | 3元 + 0.06元（育成津贴） |
| 铜牌 | D | 卖货人 | 3元 - 0.06元（给导师） |
| **总计** | | | **20元** ✅ |

**核心**：D的3元佣金，扣0.06元给导师C，实际拿2.94元

## 场景4：100人代理池，所有人都在一条链上

### 初始状态

```
金牌(1人) → 银牌(5人) → 铜牌(20人) → 普通(74人)
```

**问题**：佣金怎么分？

假设订单是最后74人中的一个人卖出的：

| 人群 | 人数 | 单人佣金 | 总佣金 |
|------|------|----------|--------|
| 金牌 | 1 | 10元 | 10元 |
| 银牌 | 5 | 6元 | 30元 |
| 铜牌 | 20 | 3元 | 60元 |
| 普通 | 74 | 1元 | 74元 |
| **总计** | **100人** | | **174元** ❌ |

**问题**：佣金超支174元，远超20元！

**结论**：不可能所有人都在一条链上！

## 场景5：100人代理池，树状结构

### 假设结构

```
金牌A
├── 银牌B1
│   ├── 铜牌C1
│   │   ├── 普通20人
│   ├── 铜牌C2
│   │   ├── 普通20人
│   └── 普通10人
├── 银牌B2
│   ├── 铜牌C3
│   │   ├── 普通20人
│   └── 普通10人
└── 银牌B3
    └── 普通20人
```

**总计**：1+3+6+90 = 100人

### 场景5.1：普通D卖货100元

**推荐链**：A → B1 → C1 → D

**佣金分配**：
- A（金牌）：10元
- B1（银牌）：6元
- C1（铜牌）：3元
- D（普通）：1元
- **总计：20元** ✅

### 场景5.2：C1升级到银牌

**推荐链**：A → B1 → 银牌C1 → 普通D → 客户

**D卖货100元**（按方案D）：

| 代理人 | 等级 | 佣金计算 |
|--------|------|----------|
| A | 金牌 | 10元 |
| B1 | 银牌 | 6元 |
| C1 | 银牌 | 6元 - 0.12元（给B1） = 5.88元 |
| B1（导师） | | +0.12元（育成津贴） |
| D | 普通 | 1元 |
| **总计** | | **20元** ✅ |

**注意**：
- C1升级后，佣金从3元变成6元
- B1作为导师，抽C1佣金的2%（0.12元）
- B1总收益：6元 + 0.12元 = 6.12元（比升级前多0.12元）

## 核心问题总结

### 问题1：推荐链深度限制

如果所有人都严格按4级分层，那么：
- 任何订单最多分给4个人
- 总佣金：10+6+3+1 = 20元 ✅

### 问题2：下级升级后，上级收益变化

**升级前**：
- 上级拿基础佣金（铜牌3元）

**升级后**（方案D）：
- 上级拿基础佣金（铜牌3元） + 育成津贴（下级佣金×2%）
- 上级总收益：3元 + 0.12元 = 3.12元 ✅ 增加！

### 问题3：育成津贴从哪里出？

**方案D：从下级佣金扣**
- 下级赚6元，扣2%给导师，实际拿5.88元
- 导师拿基础佣金3元 + 育成津贴0.12元 = 3.12元
- 总佣金池：20元 ✅

---

## 推荐方案：方案D

### 核心逻辑

1. **基础佣金按推荐链分**：金牌10%、银牌6%、铜牌3%、普通1%
2. **下级升级后，导师拿育成津贴**：下级佣金×2%
3. **育成津贴从下级佣金里扣**：不影响总佣金池

### 收益对比（C1升级前 vs 升级后）

| 人物 | 升级前 | 升级后 | 变化 |
|------|--------|--------|------|
| A（金牌） | 10元 | 10元 | 不变 |
| B1（银牌） | 6元 | 6元 + 0.12元 = 6.12元 | +0.12元 ✅ |
| C1（铜牌→银牌） | 3元 | 6元 - 0.12元 = 5.88元 | +2.88元 ✅ |
| D（普通） | 1元 | 1元 | 不变 |
| **总计** | **20元** | **20元** | **0元** ✅ |

**结论**：
- 上级收益增加（+0.12元）
- 下级收益大幅增加（+2.88元）
- 公司收益不变（80元）
- **三赢！**
