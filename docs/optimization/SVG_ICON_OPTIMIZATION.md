- ✅ 钱包主体使用渐变填充
- ✅ 装饰金币使用 `#FFD700` 并调整透明度
- ✅ 增强白色高光和阴影层次
- ✅ 保持简洁的钱包几何设计

**效果**: 温暖、精致、有层次感

## 配色系统统一

### 全局色调
```scss
主色调:
- 琥珀金: #C9A962 (主要强调色)
- 金色: #FFD700 (高光、亮点)
- 青铜: #D4A574 (次级强调)
- 铜: #B8860B (金属质感)
- 深棕: #3D2914 (描边、深色)

渐变方向:
- 琥珀金 → 青铜 (温暖、高级)
- 金色 → 琥珀金 (明亮、温暖)
- 三色渐变 (层次丰富)
```

### 移除的冷色调
- ❌ `#06D6A0` (青绿)
- ❌ `#05B384` (蓝绿)
- ❌ `#047A5F` (深蓝绿)
- ❌ `#FFB085` (单色橙黄 - 升级为渐变)

## 设计原则遵循

✅ **配色统一**: 所有图标使用温暖的金/铜/琥珀金系统
✅ **渐变应用**: 每个图标都有适当的渐变填充或描边
✅ **视觉深度**: 通过透明度变化、渐变层次、滤镜光晕
✅ **几何简洁**: 保持原始设计的简洁几何形状
✅ **可缩放性**: viewBox和stroke-width确保所有尺寸下清晰
✅ **品牌一致**: 符合精酿啤酒的温暖高级美学

## 技术实现

### SVG技术特性
1. **linearGradient**: 创建平滑的金属质感渐变
2. **opacity控制**: 通过透明度变化增加深度
3. **filter滤镜**: 皇冠图标使用高斯模糊光晕
4. **stroke-width**: 保持2-3px的描边粗细
5. **viewBox**: 统一使用 0 0 48 48 或 0 0 32 32

### 优化效果
- **视觉一致性**: 所有图标色调统一为温暖金属色
- **深度感**: 渐变和透明度营造立体感
- **质感提升**: 金属渐变模拟真实材质
- **精致度**: 微妙的光晕和高光
