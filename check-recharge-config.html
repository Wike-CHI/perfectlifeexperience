充值配置检查工具

请在微信小程序开发者工具的云开发控制台中运行以下操作:

步骤1: 查询当前配置

db.collection('system_config')
  .where({ type: 'commission_config' })
  .get()
  .then(res => {
    console.log('当前配置:', res.data);
    if (res.data[0]?.rechargeOptions) {
      console.table(res.data[0].rechargeOptions);
    }
  });

步骤2: 如果有赠送金额,运行此脚本清除

db.collection('system_config')
  .where({ type: 'commission_config' })
  .update({
    data: {
      rechargeOptions: [
        { amount: 100, gift: 0 },
        { amount: 200, gift: 0 },
        { amount: 500, gift: 0 },
        { amount: 1000, gift: 0 },
        { amount: 2000, gift: 0 }
      ],
      updateTime: new Date()
    }
  })
  .then(res => {
    console.log('✅ 充值配置已更新为无赠送模式');
    console.log('更新结果:', res);
  });

步骤3: 验证更新结果

db.collection('system_config')
  .where({ type: 'commission_config' })
  .field({ rechargeOptions: true })
  .get()
  .then(res => {
    const opts = res.data[0]?.rechargeOptions || [];
    const hasGift = opts.some(o => o.gift > 0);
    console.log('充值档位数量:', opts.length);
    console.log('是否包含赠送:', hasGift ? '❌ 是(需要清除)' : '✅ 否');
    console.table(opts);
  });

检查清单:

  • ✅ 前端代码已修复(recharge.vue 初始化逻辑)
  • ✅ 管理后台默认值已修复(无赠送)
  • ✅ 后端默认配置已修复(defaultConfig.js)
  • ⏳ 数据库配置需要验证
